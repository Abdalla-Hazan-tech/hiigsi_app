# Task Manager System

## 📌 Description
This is a web-based task manager system that helps users:
- Add tasks
- Update tasks
- Delete tasks
- Track progress

## 🎯 Problem Solved
Helps students manage academic tasks and deadlines.

## 🛠️ Technologies Used
- Node.js
- Express.js
- MongoDB
- React.js

## 🚀 How to Run

### Backend
cd backend
npm install
npm start

### Frontend
cd frontend
npm install
npm start
